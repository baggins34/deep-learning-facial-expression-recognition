Recognition of emotions from a gray-scale image. 
"fer2013.csv" is used to train the model.
CNN is used for 7 classes:
- 0: Angry 
- 1: Disgust 
- 2: Fear
- 3: Happy
- 4: Sad
- 5: Surprise
- 6: Neutral
